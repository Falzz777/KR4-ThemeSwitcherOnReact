# Theme Switcher - React приложение

React приложение для переключения между светлой и тёмной темами.

## Функциональность

- Переключение между светлой и тёмной темами
- Автосохранение выбранной темы в localStorage
- Плавные анимации перехода
- Адаптивный дизайн
- Современный UI/UX
