
**Инструкции по запуску:**

1. Создайте папку проекта и файлы как указано выше
2. Установите зависимости: `npm install`
3. Запустите приложение: `npm start`
4. Откройте браузер по адресу: `http://localhost:3000`

Приложение готово к использованию и демонстрирует полный функционал переключения тем с сохранением состояния.
